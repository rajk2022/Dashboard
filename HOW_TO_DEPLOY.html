Get a Permanent Link — Step by Step

Deploy your Training Dashboard online · Free · Takes ~10 minutes

1

Create a Free GitHub Account FREE

GitHub is where you'll store your dashboard files. It's free and takes 2 minutes.

  • Go to github.com → click Sign up
  • Choose any username (e.g., training-dashboard)
  • Verify your email
2

Create a New Repository

A repository = a folder on GitHub to store your app files.

  • Click the + icon (top right) → New repository
  • Name it: training-dashboard
  • Set to Public
  • Click Create repository
3

Upload These 3 Files from the ZIP you downloaded

On your new GitHub repo page, click uploading an existing file.

Drag and drop these 3 files:

📄 app.py
📄 dashboard_data.json
📄 requirements.txt
  • Click Commit changes (green button at bottom)
💡 Your files are now on the internet — but the app isn't live yet. Next step does that.
4

Deploy on Streamlit Community Cloud FREE

  • Go to share.streamlit.io
  • Click Sign in with GitHub — use the same account
  • Click New app
  • Fill in:
    Repository: your-username/training-dashboard
    Branch: main
    Main file: app.py
  • Click Deploy!
  • Wait ~2 minutes for it to build
✅ You now have a permanent link! It looks like:

https://your-username-training-dashboard-app-xxxx.streamlit.app
5

🎉 Share the Link with Your Manager

Copy the URL from Streamlit and send it. That's it. Your manager can:

  • View the full dashboard on any device, any browser
  • Click ✏️ Edit Data tab → edit numbers directly
  • Click 💾 Save Changes → charts update instantly
  • Switch back to 📊 Dashboard tab to see new charts
✏️

How to Update Data Later

Option A — Easiest: Use the Edit Data tab in the dashboard directly. No files, no uploads.


Option B — Bulk update from Excel:

  • Update your Dashboard.xlsx (Sheet1)
  • Run this one command in the same folder:
python regenerate.py
  • Upload the new dashboard_data.json to GitHub (drag & drop, replace old file)
  • Dashboard updates automatically within 30 seconds

Your Permanent Dashboard URL

https://[yourname]-training-dashboard-app.streamlit.app

Send this link to your manager. No login needed to view. Works on mobile too.
